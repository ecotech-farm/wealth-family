Cài đặt

Đồng bộ gia đình (Firebase)
Hướng dẫn thiết lập (1 lần duy nhất)
  1. Vào console.firebase.google.com
  2. Tạo project mới (tên tùy ý, VD: "wealth-family")
  3. Vào Build > Realtime Database > Create Database
  4. Chọn location asia-southeast1 > Start in test mode
  5. Copy URL database (VD: https://xxx.firebaseio.com)
  6. Vào Project Settings (icon bánh răng) > General > copy API Key
  7. Điền 3 trường bên dưới > bấm Kết nối
